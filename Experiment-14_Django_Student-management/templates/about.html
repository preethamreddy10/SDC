About CHONY Student Management

Project Overview

CHONY Student Management System is a comprehensive web application built with Django framework to streamline student data management in educational institutions.

Key Objectives

  • Provide a centralized platform for student information management
  • Streamline administrative processes and reduce paperwork
  • Enable efficient tracking of academic progress
  • Facilitate quick access to student records
  • Ensure data security and user authentication

Technical Features

  • Backend: Django 5.2 with Python
  • Database: SQLite for development
  • Frontend: Bootstrap 5 with responsive design
  • Authentication: Django's built-in user authentication
  • Forms: Django Forms with validation
  • Admin Interface: Django Admin for data management

Student Management Features

  • Student registration and profile management
  • Academic information tracking (CGPA, department, year)
  • Search and filter capabilities
  • Department-wise organization
  • Secure data storage and retrieval
